# MugaMuchu

## 対象OS
- Windows

## インストール
1. Releaseから"MugaMuchu.zip"をダウンロードして、展開

## 起動
1. 展開されたフォルダ内の"MugaMuchu.exe"を実行

## 遊び方
### カメラ(視点)操作
- 右クリック長押ししながらドラッグ
    - キャラがY軸方向に回転
- マウスホイールを動かす
    - キャラを対象にズームイン＆ズームアウト
- マウスホイール長押ししながらドラッグ
    - カメラ移動

### キャラ操作
- Jキー押下
    - 右手がランダムに動く
- Fキー押下
    - 左手がランダムに動く
